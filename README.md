# Data Cleaning Project

## Overview
This project demonstrates a complete data cleaning process using SQL on a real-world layoffs dataset. The objective is to transform raw, inconsistent data into a clean and structured dataset that is ready for analysis.

The project showcases common SQL data cleaning techniques that are frequently used by data analysts and data scientists in real-world scenarios.

---

## Dataset
The dataset contains information about company layoffs, including:

- Company
- Industry
- Location
- Total layoffs
- Percentage laid off
- Date
- Stage
- Country
- Funds raised

**File included:**
- `layoffs.csv`

---

## Project Objectives

The cleaning process includes:

- Removing duplicate records
- Standardizing inconsistent data
- Handling NULL and missing values
- Formatting date columns
- Correcting text inconsistencies
- Removing unnecessary columns
- Preparing the dataset for analysis

## Project Workflow

1. Import the raw dataset.
2. Create a staging table.
3. Remove duplicate records.
4. Standardize text values.
5. Handle missing and NULL values.
6. Convert data types where necessary.
7. Remove unnecessary columns.
8. Produce a cleaned dataset ready for analysis.
